## 👋 Welcome to hasura 🚀

Hasura - GraphQL engine for PostgreSQL

## 📋 Description

GraphQL engine for PostgreSQL

## 🚀 Services

- **app**: Hasura (`hasura/graphql-engine:latest`)

## 📦 Installation

```shell
composemgr install hasura
```

## 🔧 Configuration

```shell
TZ=America/New_York
BASE_HOST_NAME=hasura.example.com
```

## 🌐 Access

- **Hasura**: http://localhost:8080

## 📂 Volumes

- `./rootfs/data/hasura` - Application data

## 🔐 Security

- Change default passwords
- Configure HTTPS with reverse proxy
- Regular backups
